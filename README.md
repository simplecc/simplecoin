What is Simplecoin?
----------------

Simplecoin is a lite version of Bitcoin using scrypt as a proof-of-work algorithm.
 - 1.5 minute block targets
 - subsidy halves in 1000k blocks
 - ~50 million total coins
 - 25 coins per block
 - 100 blocks to retarget difficulty
